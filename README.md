# RedBook

RedBook is a privacy-first personal reading tracker for:

- books you own
- books you want to read
- books you are currently reading
- books you finished

This repository currently contains an **MVP product specification and implementation plan** for a modern Linux desktop app built with **GTK 4 + Libadwaita**.

## MVP scope

A strong MVP includes:

1. Local library database
2. Shelves management
3. Add/edit book flows
4. Metadata lookup (Open Library + Google Books)
5. Local cover cache
6. Finished date tracking
7. Backup/restore (including encrypted backup file)

See [`docs/mvp-spec.md`](docs/mvp-spec.md) for full details.
