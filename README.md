# RedBook v5

Fixes:

- Book detail window uses the full space with a split layout.
- Categories/shelves sidebar is collapsible.
- Book cards are unified to the same size across shelves.
- Dashboard stats are smaller and fit in one row.
- Includes .deb installer with Ubuntu launcher integration and hi-res SVG icon.

Install:

```bash
sudo apt install ./redbook_0.5.0_all.deb
```
